core / makeup-roving-tabindex

Implements a roving tab index on a collection of elements. Use arrow keys to navigate.

For demonstration purposes, aria-disabled items are greyed out and hidden items have strike-through (in practice they would be visually hidden). Both are included in the index but skipped during navigation.
